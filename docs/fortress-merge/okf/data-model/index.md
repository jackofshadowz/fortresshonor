# Data Model

The recovered C# data structures underpinning the game, from the IL2CPP type dump.

* [Enums](enums.md) - rarity, tier, unit types, resources, rewards, card types, buffs, game mode
* [Card Data](card-data.md) - the unified card abstraction across units/buildings/heroes
* [Unit & Enemy Roster](unit-roster.md) - the full cast of player units, enemies, and bosses
